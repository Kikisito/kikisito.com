- Orquestación y procesamiento de datos utilizando Apache Airflow y Apache Nifi.

- Almacenamiento de datos on-premise.
- Implementación de un espacio de datos federado.
- Despliegue de infraestructura utilizando Kubernetes para garantizar disponibilidad y escalabilidad.

Durante este periodo colaboré con el equipo de Shaidraig para crear un videojuego web, combinando AngularJS con PixiJS como motor de renderizado.

- Desarrollo de componentes reusables en AngularJS para la lógica de juego y la interfaz de usuario.
- Integración de PixiJS para renderizar escenas, animaciones y sistemas de partículas optimizados para navegadores modernos.
